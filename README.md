# CampusApp

A smart campus productivity app for Nigerian university students.

## Features (In Development)
- User authentication
- Course management
- Study session tracker
- GPA calculator
- Expense tracker
- Personal dashboard

## Tech Stack
- Python / Flask
- SQLAlchemy + Flask-Migrate
- Flask-Login / Flask-WTF
- PostgreSQL (production) / SQLite (development)
- Tailwind CSS

## Setup
```bash
git clone https://github.com/auwalcodes2007/campus_app
cd campus_app
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
